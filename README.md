# AAE_4011_Project
The grand goal for this project is to train a DQN that enables the agent to find the waypoint regardless of the initial position of the agent or the waypoint. The agent (be it a drone or similar UAV) will be randomly placed and can move freely within a rectangular boundary. There will be only one waypoint during training, and its position will also be randomly initialized. The agent should reach the waypoint within a certain radius while not exceeding the time and speed limit or crossing the boundary. When that happens, the agent will have “found the waypoint.”
